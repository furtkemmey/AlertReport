# AlertReport

## 概述
-  抓取災害警示的公開資料平台，並顯示在tableView.
-  NCDR
[https://alerts.ncdr.nat.gov.tw/](https://alerts.ncdr.nat.gov.tw/ "NCDR")
-  語音播報災害內容
-  地圖顯示災害區域
## 開發環境
-  macOS High sierra 10.3.
-  Xcode 9.2.

## 內容
### UIKit
- TabBarController
- NavagationController
- TableView
- TableViewCell
- MapView
